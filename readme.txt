SPEEDRON Speed Test is totally FREE plugin. Let your users check their Internet connection speed with this pretty tool.

== Description ==

SPEEDRON Speed Test is provided by Speedron.com for free. The plugin is dedicated to give an opportunity to
measure download and upload Internet connection speed by Wordpress users. The speedtest plugin works as follows:

* the best server near from the user is selected when the plugin loads
* then the plugin is waiting for the start button to be pressed and when it is...
* first the download speed is measured
* second the upload speed is measured
* the results are provided in Mb/s (Megabits per second)
* user can restart the test if he wants

The plugin uses Cdnetx servers network and provide global coverage for accurate results.

== Installation ==

1. Extract the `speedron-speed-test` folder from the zip package and upload the `spedron-speed-test` folder (not just the files in it!) to the `/wp-content/plugins/` directory or...
1. Upload the zip package directly in Wordpress site (go to: Plugins -> Add new -> Upload) or...
1. Find `speedron speed test` in Wordpress site Search (go to: Plugins -> Add new -> Search)
1. Activate the plugin through the 'Plugins' menu in WordPress and then the widget will be available.
1. Add the SPEEDRON Speed Test widget to your sidebar.
1. Configure plugin's sidebar title.
1. You can optionally change the plugin width saving the proportions. The default width is 160px and the range is 160-300px.

== Frequently Asked Questions ==

= Is it possible to test to my server? =

Currently no. We are working on this feature and will be available soon.

= How is the best server selected? =

The best server is selected based on user's location using GeoIP engine.
